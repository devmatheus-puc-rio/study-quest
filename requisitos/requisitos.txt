Requisitos Funcionais:

[RF1] O sistema deve permitir que uma pessoa realize cadastro no site.

[RF2] O sistema deve permitir que um Usuário acesse sua conta.

[RF3] O sistema deve permitir que um Usuário veja seus planos.

[RF4] O sistema deve permitir que um Usuário crie um plano de estudos.

[RF5] O sistema deve permitir que um Usuário edite ou exclua seus planos de estudo.

[RF6] O sistema deve permitir que um Usuário compartilhe seus planos de estudo com outros usuários.

[RF7] O sistema deve permitir que um Usuário visualize sugestões automáticas de conteúdo com base nos planos de estudo criados.

[RF8] O sistema deve permitir que um Usuário crie quizzes personalizados relacionados aos seus planos de estudo.

[RF9] O sistema deve permitir que o Usuário receba notificações de prazos e lembretes para os planos de estudo.


Requisitos Não Funcionais:

[RNF1] O sistema deverá gerar o resultado da busca por curso em menos de 1 segundo.

[RNF2] O sistema deve ser compatíel em dispositivos móveis.

[RNF3] O sistema deve suportar pelo menos 10 usuários simultâneos sem perda significativa de desempenho.

[RNF4] O sistema deverá ter uma interface de usuário intuitiva e fácil de usar para pessoas com diferentes níveis de experiência tecnológica.

[RNF5] O site deve seguir as diretrizes de acessibilidade, garantindo que pessoas com deficiência possam utilizar o sistema.

[RNF6] O sistema deve ser seguro e proteger os dados do usuário com criptografia de senha.

[RNF7] O tempo de resposta do servidor para carregar uma página deve ser de no máximo 2 segundos.
